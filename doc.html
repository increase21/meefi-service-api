MOBILE TRANSFER API

Name Enquiry: Confirm a user's account details

GET: http(s)://base-url/v1/api/name-enquiry
Header-> Authorization: Bearer {api_key}
Query Parameters Type Status Description
fin_account string required Account number to confirm
fin_code string required The code that references the bank/financial institution See code list here
country_code string required See list of supported countries

Response data on success

            
   {
      "status": "ok",
      "code":"CS200",
      "data":{
         "firstname": "Test",
         "lastname": "MFS Benin",
         "gender": "MALE",
         "fin_account": "09094523232",
         "fin_name": "mtn"
       }
   }

Transfering funds to another user

GET: http(s)://base-url/v1/api/transfer
Header-> Authorization: Bearer {api_key}
Fields Type Status Description
origin_fin_code String required Origin bank code See list of supported
origin_fin_account String required Origin account number
destination_fin_code String required Destination bank code See list of supported
destination_fin_account String required The destination account number
receiver_firstname String required
receiver_lastname String required
country_code String required See list of supported countries
amount String required
description String optional
app_platform String required Value must be ussd or mobile-app
external_id String optional A unique id to reference your request.

Response data on success

            
{
   "status": "ok",
   "code":"CS200",
   "data":{...}
}
